# single_character — the simplest renderable scene

A one-shot, two-second scene with a single character (placeholder rect parts).
This is the Phase 2D smoke target — the first end-to-end mp4 anima can make.

```bash
cd examples/single_character
anima sync .
anima render .
# → output/main.mp4 (~12 KB, 2s @ 24fps)
```
