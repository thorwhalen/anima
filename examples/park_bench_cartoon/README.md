# park_bench_cartoon — the v0.1 demo target

A 45-second 2D cutout cartoon, two characters on a park bench, dialogue with
lip-sync, one slow camera push-in.

**Phase-1 status:** skeleton only. The `scene.md` is hand-authored and
`anima validate .` should report it clean. The `ir/scene.json` is generated by
running `anima sync .` (or `anima validate .`, which calls sync first).

**The full demo lights up in Phase 4** once the cutout adapter (Phase 2) and
audio pipeline (Phase 3) are in. At that point this directory will round-trip
end-to-end: edit `scene.md` → `anima render .` → `output/park_bench.mp4`.

## Try it now (Phase 1)

```bash
cd examples/park_bench_cartoon
anima sync .       # regenerate ir/scene.json from scene.md
anima validate .   # schema + semantic validation
```
